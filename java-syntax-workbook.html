Java Syntax Study Workbook

Color workbook for learning Java syntax through mini-problems, short sessions, and repeatable review. Built for low-friction ADHD study.

10 min learn 15 min mini-problems 10 min old review 5 min mistake log

Study Rule

One topic. Tiny reps. Stop while the session still feels clean.

How To Use It

  • Pick one syntax topic only.
  • Copy one example with notes.
  • Rewrite it from memory.
  • Solve 2 mini-problems.
  • Redo 1 old mini-problem.
  • Write what you forgot before stopping.

Move On Rule

  • You can write the syntax without notes.
  • You solved at least 4 of 5 mini-problems cleanly.
  • You know when to use it in a DSA problem.

30-Day Roadmap

Days 1-5

Variables, arithmetic, if/else, for, while, methods.

Days 6-10

Arrays, strings, review and speed reps.

Days 11-15

Nested loops, brute force thinking, ArrayList.

Days 16-20

HashSet, HashMap, frequency patterns, Two Sum.

Days 21-25

Deque as stack and queue, PriorityQueue.

Days 26-30

Recursion, 2D arrays, sorting, binary search.

Mini-Problem Bank

Solve tiny problems until the syntax starts to feel automatic. Keep them short. Repeat old ones often.

Core Basics

  • Print your age.
  • Add two numbers.
  • Rectangle area.
  • Swap two values.
  • Minutes to seconds.

If / Else

  • Even or odd.
  • Positive, negative, or zero.
  • Max of two numbers.
  • Is vowel.
  • Score to grade.

Loops

  • Print 1 to 10.
  • Print 10 to 1.
  • Sum 1 to n.
  • Multiplication table.
  • Count digits.

Methods

  • `add(a, b)`
  • `isEven(n)`
  • `max(a, b)`
  • `square(n)`
  • `countDigits(n)`

Arrays

  • Print all elements.
  • Find sum.
  • Find maximum.
  • Find target index.
  • Count evens.

Strings

  • Count vowels.
  • Reverse a string.
  • Check palindrome.
  • Count uppercase letters.
  • Count char frequency.

Collections Mini-Problems

SyntaxMini-Problems
